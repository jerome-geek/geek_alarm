## 자주 사용하는 pm2 명령어

```bash
# 실행중인 프로세스 확인
$ pm2 list

# 시작
$ pm2 start {node file}

# 중지
$ pm2 stop {id or App name}

# 삭제
$ pm2 delete

# 상세정보
$ pm2 show

# 재시작
$ pm2 restart

# 종료
$ pm2 kill

# id앱의 로그 확인
$ pm2 logs {id}
```
